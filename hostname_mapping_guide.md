# 🎯 호스트명 매핑 가이드

## Replit vs 가비아 호스트명 차이점

### Replit에서 표시되는 것:
- **전체 도메인**: www.jasoai.co.kr
- **호스트명**: www.jasoai.co.kr (전체 형태)

### 가비아에서 입력해야 하는 것:
- **호스트 필드**: **www** (서브도메인 부분만)
- **도메인**: jasoai.co.kr (자동으로 붙음)

## 📝 올바른 가비아 설정

```
호스트: www
타입: A
값: 34.111.179.208
TTL: 300

호스트: www  
타입: TXT
값: replit-verify-xxx...
TTL: 300
```

## ⚠️ 흔한 실수들

### 잘못된 입력:
❌ 호스트: www.jasoai.co.kr (전체 도메인)
❌ 호스트: jasoai.co.kr (루트 도메인)
❌ 호스트: @ (루트 기호)

### 올바른 입력:
✅ 호스트: www (서브도메인만)

## 🔍 확인 방법

가비아 DNS 관리에서:
1. 호스트 필드에 "www"만 입력되어 있는지 확인
2. 완성된 레코드가 "www.jasoai.co.kr"로 표시되는지 확인
3. TTL이 300-600초로 설정되어 있는지 확인
